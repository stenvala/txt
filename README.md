# TXT

Nice mobile friendly UI to YLE's tekstitelevisio

Technologies

- Angular 7
- Netcore 3.1

## Instructions for developers

```bash
# Initialize
chmod 755 install.sh
./install.sh
# Activate venv
. venv/bin/activate
# Start services
python start_services.py
# Stop services
python stop_services.py
# Build
python build.py
# Deploy
php deploy_backend.php
php deploy_ui.php
```
